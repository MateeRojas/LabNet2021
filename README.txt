En el presente trabajo se hace entrega de una aplicación de consola dividia en capas. La misma se conecta a la base de datos Northwind
y a través de linq se hacen las diferentes querys pedidas.
Aclaraciones:
En la mayoría de los casos se consideró como dato reelevante a mostrar únicamente el id y el nombre.
Por eso mismo se implemtentaron las funciones para mostrar datos en la capa de lógica.
En algunas querys especificas se se mostraron los datos abajo mismo de la consulta ya que eran particulares y no ameritaban una función
aparte asi como tambien se agregó que mostraran el dato que demuestre que cumplían lo pedido.
